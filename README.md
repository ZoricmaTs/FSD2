# FSD2
Верстка UI-Kit по FIGMA

##Развёртывание
***
Клонировать
```
git clone https://github.com/ZoricmaTs/FSD2.git
```
Установка
***
```
npm i
```
Запустить parcel
```
npm run dev
```
Запустить production build
```
npm run build
```
